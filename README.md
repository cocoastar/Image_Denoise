# Image_Denoise  
图像去噪的目标是从受噪声干扰的退化图像中尽可能恢复原始的真实图像，是图像进行后续处理的关键一步。在本文中，我们基于给定的REDNet和DnCNN两个典型深度图像去噪模型，在BSD68数据集上实现了图像去噪功能。基于课设要求，
## How to run  
### 环境配置要求  
* PyTorch
* OpenCV for Python
* HDF5 for Python
* Scikit-Image
### 运行训练函数  
`<
python train.py \
  --preprocess True \
  --num_of_layers 17 \
  --mode S \
  --noiseL 25 \
  --val_noiseL 25>`
